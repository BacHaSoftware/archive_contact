## Usage

Module allows to archive contact when related processes have been completed.


#### Featured Highlight:

üåü <code>Simple Setup</code>: After installing module, archive contact feature is added automatically without any further configuration.

üåü <code>Reduce Process</code>: Contacts that are no longer needed are automatically archived, saving time and simplifying process.
